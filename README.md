### Проект учебный "Телеграм дома".

Учебный проект для курса "middle frontend" от Яндекс.Практикума

https://tgdoma.netlify.app

### Запуск проекта

Запуск в режиме разработчика

```sh
    npm run dev
```

Сборка билда

```sh
    npm run build
```

Сборка билда и запуск проекта

```sh
    npm run start
```
